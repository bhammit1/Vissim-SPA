This folder contains all python scripts that can be used to run Vissim models through python.

It is HIGHLY recommended to run your Vissim models for MOEs using this script. For best functionality, pair this python script with our MOE Spreadsheet Template.

It will save time, resources, and avoid mistakes. 

---------------------------------------------------------------------------------------------------
Steps to Run:
* Open "main.py" script in Spyder
	* Download Anaconda Python 3.8, 64Bit for Windows: https://www.anaconda.com/products/individual
* Update project inputs
* Press play and wait for your email :)
